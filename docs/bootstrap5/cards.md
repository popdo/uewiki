# 卡片

卡片是一个灵活的、可扩展的内容容器，它包括`headers`和`footers`的选项、各种内容、上下文背景颜色和强大的显示选项。如果你熟悉Bootstrap3，那么card将取代我们旧的面板、和缩略图。与这些组件类似的功能都可以使用card实现。

通过在`<h*>`标签中添加`.card-subtitle`来使用副标题。如果将`.card-title`和`.card-subtitle`项放在`.card-body`项中，那么卡片标题和副标题就会很好地对齐。

## 卡片类对照表

| 类名称                | 说明                                                                                                     |
|----------------------|---------------------------------------------------------------------------------------------------------|
| `.card-group` (外部容器) | 卡片组：将该容器内的.card编组                                                                             |
| `.card`              | 卡片自身类名                                                                                               |
| `.card-header`       | 卡片头部                                                                                                  |
| `.card-body`         | 卡片主体                                                                                                  |
| `.card-footer`       | 卡片尾部                                                                                                  |
| `.card-title`        | 卡片标题                                                                                                  |
| `.card-subtitle`     | 卡片副标题                                                                                                |
| `.card-text`         | 卡片文字                                                                                                  |
| `.card-link`         | 卡片链接                                                                                                  |
| `.card-img`          | 卡片图片                                                                                                  |
| `.card-img-top`      | 卡片顶部图片                                                                                               |
| `.card-img-bottom`   | 卡片底部图片                                                                                               |
| `.card-img-overlay`  | 将该容器漂浮在card-img上面(`<div class="card"><img class="card-img"><div class="card-img-overlay"></div></div>`) |
| `.text-muted`（全局类）| 一般应用在`<small>`上，常用在表单补充说明、文档日期等文本上                                                      |
